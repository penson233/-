# static_log

this is for nginx.log.
it can help you to static ip,request,ip's amount,request's amount.

you must install Python third-party library------openpyxl,prettytable brefore you use it!

帮助你分析nginx的日志文件，将以excel表格形式存储。
需要安装python第三方库
prettytable 
openpyxl

会以excel形式保存
